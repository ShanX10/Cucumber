# Cucumber

## Overview

This repository contains Behavior-Driven Development (BDD) automation projects created using Cucumber, Selenium WebDriver, and Java. The projects demonstrate how feature files, step definitions, and test runners work together to automate web application testing.

## Technologies

* Java
* Selenium WebDriver
* Cucumber
* Gherkin
* TestNG

## Topics Covered

* Feature Files
* Step Definitions
* Test Runners
* Login Scenarios
* Search Scenarios
* Assertions
* Multiple Test Scenarios
* BDD Framework Design

## Purpose

The objective of this repository is to demonstrate the implementation of Behavior-Driven Development practices for automated software testing.
